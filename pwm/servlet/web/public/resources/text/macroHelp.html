Macros can be used at various places in the configuration.

The following macros are available:

@LDAP:<name>@ Replace with LDAP value of <name>
@LDAP:DN@ Replace with LDAP value of user's Distinguished Name
@User:PwExpireTime@ Time user's password will expire in PWM default format.
@User:PwExpireTime:<pattern>@ Time user's password will expire where <pattern> is a SimpleDateFormat pattern
@User:DaysUntilPwExpire@ Number of days until the user's password will expire
@User:ID@ User's UserID (if authenticated)
@User:Email@ User's Email Address
@User:Password@ User's current password (if authenticated)
@InstanceID@ Instance ID of the application
@CurrentTime@ Current time in default format.
@CurrentTime:<pattern>@ Current time where <pattern> is a SimpleDateFormat pattern
@SiteURL@ URL of the site (http://www.example.com/pwm)
@SiteHost@ Hostname of the site (www.example.com)
@RandomChar:<Length>:<CharacterList>@ Random characters, where <Length> is the length of the string (or 1 if not included) and <CharacterList> is the list of characters to be used as random characters (or upper/lower ASCII characters and digits if not included). Examples: @RandomChar@ @RandomChar:16:0123456789@ @RandomChar:5@

To use a macro, place the macro into the configuration value such as:
Dear, @LDAP:givenName@, Thank you for your activity today, @CurrentTime:EEE, MMM d, yyyy@.

This macro will then be expanded to:
Dear, John, Thank you for your activity today, Wed, Jul 4, 2001.
